## Day_00 - Основы SQL

В этом проекте созданы SQL-запросы с использованием внутренних подзапросов в предложениях FROM и SELECT, а также отфильтрованы данные по диапазону дат. Отработаны навыки работы с условиями выбора, сортировки по нескольким полям с разными направлениями и структурирования запросов для получения необходимой информации.

1. [**Задание 00**](src/ex00/day00_ex00.sql): Базовый SELECT с условием WHERE;
2. [**Задание 01**](src/ex01/day00_ex01.sql): SELECT с несколькими условиями WHERE и сортировкой ORDER BY;
3. [**Задание 02**](src/ex02/day00_ex02.sql): Использование операторов сравнения и ключевого слова BETWEEN;
4. [**Задание 03**](src/ex03/day00_ex03.sql): SELECT DISTINCT с условиями по датам и OR;
5. [**Задание 04**](src/ex04/day00_ex04.sql): Конкатенация строк и форматирование вывода в SELECT;
6. [**Задание 05**](src/ex05/day00_ex05.sql): Подзапросы в SELECT;
7. [**Задание 06**](src/ex06/day00_ex06.sql): Использование CASE в SELECT для условной логики;
8. [**Задание 07**](src/ex07/day00_ex07.sql): Более сложное использование CASE для категоризации данных;
9. [**Задание 08**](src/ex08/day00_ex08.sql): Применение операции деления по модулю для выборки четных значений;
10. [**Задание 09**](src/ex09/day00_ex09.sql): Подзапросы в FROM и в SELECT для объединения данных без использования JOIN.

Для тестов необходимо запустить скрипт [model.sql](model.sql) с моделью базы данных.
