# SQL_BOOTCAMP

Репозиторий содержит комплексные решения задач по SQL, охватывающие путь от базовых запросов до продвинутой аналитики. Здесь собраны практики, охватывающие основные манипуляции с данными, агрегацию и оптимизацию запросов.
